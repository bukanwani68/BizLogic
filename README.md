# BizLogic.co.id — Peer-to-Peer Learning Platform Mockup 🚀

**BizLogic.co.id** is an interactive Single Page Application (SPA) prototype designed to solve a critical university pain point: bridging the gap between frantic exam preparation and syllabus-accurate study materials. Built entirely on a peer-to-peer (P2P) model, it enables high-achieving senior students to share their expertise and localized summaries with junior peers from the exact same university and major.

---

## ✨ Key Features Demonstrated

*   **Adaptive Search & Filter Architecture:** A dynamic multi-stage filtering workflow allowing users to query solutions by specific Universities, Majors, and distinct Learning Methods.
*   **Diverse Study Formats:** Seamless support for modern student preferences including Offline Sessional Meetings, Live Zoom Consultations, Video-on-Demand (VoD) tutorials, and Micro-PDF Ranks/Summaries.
*   **Escrow-Locked Risk Mitigation (Pitch Feature):** Includes an interactive operational modal showcasing a student-protective escrow system. Funds are safely locked until the student confirms teaching completion, ensuring zero-risk transactions.
*   **Comprehensive Student Dashboard:** A dedicated view for users to track ongoing virtual classes, fetch active video links, and monitor verified appointment schedules.
*   **Fully Responsive Layout:** Beautifully designed using Tailwind CSS following modern UX principles—clean typography (`Plus Jakarta Sans`), glassmorphism navigation, and responsive card grids.

---

## 🛠️ Tech Stack & Architecture

*   **Frontend Framework:** [Tailwind CSS](https://tailwindcss.com/) via modern CDN injection.
*   **Typography:** Plus Jakarta Sans Google Web Fonts.
*   **Core Engine:** Vanilla JavaScript DOM Manipulation (Simulated SPA State routing via targeted element classes).
*   **Styling Architecture:** Modern CSS utility-first layouting, standard flexboxes, dynamic grid sizing, and glass-like backdrops.
